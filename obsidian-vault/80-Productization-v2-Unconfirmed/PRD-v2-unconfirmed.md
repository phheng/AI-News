# [未确认版本] PRD v2

## 1. 产品定位
Decision-first Agent Console：
“先告诉你该做什么，再给证据和执行路径。”

## 2. MVP 模块
1) Tasks：任务收敛与优先级
2) Recommendations：建议 + 置信度 + 证据
3) Approvals：审批流（Observe/Approve/Auto）
4) Insights：业务/产品/成本指标
5) Assets：Skill/Plugin/Trigger/Channel/Experience清单

## 3. 核心用户流
- Daily Triage：打开即看到今日优先动作
- Approval Flow：建议审阅→批准/拒绝→留痕
- Weekly Review：指标回看→策略调整

## 4. 功能优先级
### Must
- 任务与建议可见
- 审批可执行
- 指标可查看
- 日志可追溯

### Should
- 按卖家类型输出差异化策略
- 周报与行动闭环

### Could
- 自动化策略模板市场
- AI 可发现性外部监控

## 5. 验收标准
- 建议→审批→回写流程可走通
- 关键页面可用（非仅静态）
- API 与前端字段对齐
- 风险动作可拦截
