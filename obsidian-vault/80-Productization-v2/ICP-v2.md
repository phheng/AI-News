# ICP v2（北美版）

## 分层
1. Primary：品牌型（Private Label / Brand-led SMB）
2. Secondary：贸易型（Wholesale / Authorized Reseller）
3. Selective：铺货型（Retail/Online Arbitrage）
4. Defer：Dropshipping
5. Defer：Handmade

## 北美侧重点（相对中国卖家）
- 更强调合规、品牌资产、审计留痕
- 更重视可解释建议（evidence + confidence）
- 更关注广告效率与库存现金流联动

## 核心痛点映射
- 决策分散：工具多，结论慢
- 成本压力：ACOS/TACOS 与费率波动
- 风险压力：断货、差评、合规触发
- 执行压力：团队小、动作多、复盘弱

## 价值主张
- 10秒级单任务洞察（阶段1）
- 多工具智能编排（阶段2）
- 主动策略提醒与个性化决策（阶段3）
