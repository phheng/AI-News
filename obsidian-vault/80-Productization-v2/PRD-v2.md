# PRD v2

## 产品定位
Decision-first Agent Console（可执行决策台）

## MVP 功能
- Tasks：任务与优先级
- Recommendations：建议 + 证据 + 置信度
- Approvals：审批队列与决策闭环
- Insights：业务/产品/成本指标
- Assets：Skill/Plugin/Trigger/Channel/Experience

## 核心流程
1. 生成建议
2. 风险分级
3. 审批/执行
4. 指标回写
5. 复盘优化

## 验收标准
- 关键流程可走通（建议->审批->回写）
- 可视化页面可直接使用
- API 与页面一致
- 日志可审计
